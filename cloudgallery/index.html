Cloud Gallery

Cloud Gallery is an android app designed for TV devices. With the app, users can easily access their photos and videos on the TV by integrating multiple cloud services

Features

Cloud service media source

Access your cloud hosted photos and videos organized by albums

Built in Player

Watch your photos and videos in a media player that works with the remote

Pincode Lock

Secure access to the app by setting up a pin code

Screensaver

Configure a set of albums to show when the the TV goes to sleep
